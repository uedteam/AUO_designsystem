為商業軟體與服務
打造視覺介面設計系統

由友達設計創新研究院/數位體驗設計處開發的 AUO Design System，致力於形塑友達數位產品的設計語言，優化企業 (B 端) 及消費者 (C 端) 的使用者體驗，加速產品開發流程，並提供清晰的設計規範指引、豐富的元件庫和實用的開發資源，幫助設計師和開發者打造一致、高效能且美觀的使用者介面，為產品提供絕佳操作體驗。

什麼是設計系統

由 UI/UX 團隊的設計師與前端工程師合作開發，以 UX 體驗作為操作邏輯基準，包含設計準則與前端框架元件。內建 AUO Reshape Style 讓數位產品保持高度和諧與易用性，強化產品體驗及品牌識別度

導入效益

  • 可視化、可擴充、可複製、易調整
  • 加速商業提案，迅速回應市場
  • 大幅提升軟體操作易用性及視覺美感
  • 兼容各大開源框架 ( e.g., Tailwind, Bootstrap )

促進團隊高效協作

提供標準化的設計語言和元件庫，讓設計師與開發者可以更有效率地進行協作，減少溝通成本和誤解。開發者也可以獨立透過 Storybook 搭配範本，快速開發商業軟體操作介面。

確保設計開發一致性

提供完整的元件庫，包含 Figma 和前端對應元件，讓設計稿可以無縫轉換為程式碼，確保設計與開發的一致性。

縮短產品開發週期

商業市場瞬息萬變，AUO 設計系統協助產品團隊快速建置最小可行性系統雛形 (MVP)，有效縮短開發與驗證時程，快速取得市場先機。

滿足不同商業領域需求

AUO 設計系統除了涵蓋 AUO Reshape 規範外，更可以透過內建色彩和風格基礎系統，針對不同商業應用場景 (例如醫療、教育) 和客戶需求，靈活設計各式主題風格。

提升品牌形象與用戶體驗

AUO 設計系統協助打造直觀易用的數位產品，提升使用者體驗，進而強化品牌識別，建立良好的品牌形象。

快速回應不同商業領域客戶

AUO 設計系統打造 Design Token 的變數設定，數位產品可輕鬆切換不同的主題風格，無論是 to B 或 to C 的商業領域，皆可滿足不同的目標客群和應用場域。
