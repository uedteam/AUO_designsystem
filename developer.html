安裝說明

01.
建立開發環境

  • npx create-react-app ${my-app}
  • cd ${my-app}
  • npm start

02.
安裝相依套件

  • 開啟 package.json 設定檔
  • 新增 SASS 到 devDependencies : "sass": "^1.55.0"
  • 執行 npm install
  • 執行 npm i ded-wds

03.
引用SCSS

  • 複製 style 資料夾
  • style.zip
  • 進入點檔案( index.js )引用 globals.scss

04.
開始頁面開發

  • 打開 Storybook
  • 設定、複製、貼上
  • 程式開發
